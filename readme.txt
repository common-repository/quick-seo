== Description ==

Quick Search engine optimization (SEO) can bring a tremendous amount of high-intent traffic to your website (people who are actually in the process of searching for things
Quick seo allows you to gain better result in search engines.you can upload:- 
* tilte tag
* description tag
* canonical
* publisher tag 
* author tag
* robots.txt
* xml sitemap
* google analytics
* additional header script and style
* google and bing verification tag
* social icon
* google plus button 





== Installation ==

1. Download the plugin and extract the files
2. Upload to your wp-content/plugins/ directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Check the "Quick Seo" Tab created by this plugins for setup.
